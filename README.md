# Equi Project

## Overview
Equi is a web project dedicated to the emotional well-being of university students. The platform provides resources, tips, and a supportive community to help students maintain a healthy balance in their academic and personal lives.

## Features
- **Main Page**: An engaging introduction to emotional well-being with a clear call to action.
- **About Us**: Information about our mission, vision, and values.
- **Health and Well-being**: Practical tips and articles for maintaining emotional and physical balance.
- **Resources**: Tools for organizing tasks, reminders for breaks, and a virtual community.
- **Contact**: A simple contact form for inquiries and support.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd Equi
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```
